🚀 Opciones de publicación gratuita
⚡
Netlify Drop — Más fácil
Sin cuenta · Enlace en 30 segundos
  1. Abre app.netlify.com/drop en tu ordenador (no en móvil)
  2. Arrastra el archivo descargado a la zona azul
  3. ¡Listo! Tienes tu enlace: algo.netlify.app
💡 Para que el enlace no cambie, créate una cuenta gratis en Netlify
🐛
Glitch.com — Enlace permanente gratis
Requiere cuenta (gratis con Google)
  1. Ve a glitch.com → "Sign in with Google"
  2. Clic en "+ New Project" → "glitch-hello-website"
  3. En el panel izquierdo, abre index.html
  4. Borra todo el contenido y pega el HTML de tu archivo generado
  5. Tu URL es: tu-proyecto.glitch.me
🐱
GitHub Pages — Más profesional
Requiere cuenta GitHub (gratis)
  1. Ve a github.com → crea cuenta gratis
  2. "New repository" → ponle nombre (ej: fondo-solidario) → público
  3. Sube el archivo renombrado como index.html
  4. Settings → Pages → Branch: main → Save
  5. Tu URL: tu-usuario.github.io/fondo-solidario
